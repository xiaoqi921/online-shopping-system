# online-shopping-system
A full-stack online shopping system built with Java, Spring Boot and Vue3, and an e-commerce management system with user and admin roles.
项目介绍

Online Shopping System 是一个基于 Spring Boot + Vue3 开发的前后端分离商城系统。

系统包含两种角色：

用户(User)
管理员(Admin)

支持商品浏览、购物车、订单管理以及后台商品管理等功能。

技术栈

Backend:
Java
Spring Boot
MyBatis Plus
MySQL
JWT
Maven

Frontend:
Vue3
Vite
Pinia
Vue Router
Axios
Element Plus
